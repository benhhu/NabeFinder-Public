Welcome to NabeFinder!

  •  About the APP
  •  Slides
  •  Disclaimer

A web APP to find best neighborhood for people with disabilities using data from the American Community Survey, the Behavioral Risk Factor Surveillance System (BRFSS), and Google Places API, built with Bootstrap 3, Leaflet, MySQL, and Flask. Open source and available on GitHub.

Features
  • Provides personalized predictions on affordability and health.
  • Includes 16 individual neighborhood indicators.
  • Weights determined by users' preferences.

The data provided on this site is for informational and planning purposes only.
